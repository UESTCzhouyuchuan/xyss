# xyss
成电送水
## 项目简介
>为电子科大广大师生提供送水服务，覆盖清水河，沙河两个校区的教学楼，科研楼以及寝室等区域。
## 数据结构
>从校区，区域，楼栋，房间形成一个从根到叶的**树状结构**，因为数据是固定的使用数组较为简单些
